## 3D Car Models (GLB format)

Place your exported `.glb` files here with these exact names:

- **`car-hatchback.glb`** — Taxi by Poly by Google (CC-BY) via Poly Pizza
- **`car-truck.glb`** — Truck by KolosStudios (CC-BY) via Poly Pizza  
- **`car-suv.glb`** — SUV mesh
- **`car-sports.glb`** — Sports Car mesh

The game swaps them in after procedural spawn when you select that car.

**Other cars:** Rickshaw stays procedural. Add more by placing a `.glb` here and setting the `glb` field in `src/cars/*.js`.
